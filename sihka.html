Введение в консольный ввод/вывод в C#

В C# класс Console предоставляет основные методы для взаимодействия с консолью. Два фундаментальных метода для консольного ввода и вывода - это WriteLine и ReadLine.

Console.WriteLine

Метод Console.WriteLine используется для записи строки текста в консоль. Он может принимать строковый литерал, переменную или выражение в качестве входных данных.

Синтаксис

Console.WriteLine(значение);

Пример

using System;

public class Example
{
    public static void Main(string[] args)
    {
        Console.WriteLine("Привет, мир!"); // Выводит "Привет, мир!" в консоль
        string name = "Алиса";
        Console.WriteLine("Привет, " + name + "!"); // Выводит "Привет, Алиса!" в консоль
        int age = 30;
        Console.WriteLine($"Возраст: {age}"); // Выводит "Возраст: 30" в консоль (интерполяция строк)
    }
}

Объяснение

  • Первый оператор WriteLine выводит простой строковый литерал.
  • Второй оператор WriteLine использует конкатенацию строк для объединения строкового литерала с переменной.
  • Третий оператор WriteLine использует интерполяцию строк (используя символ $ перед строкой) для внедрения значения переменной age непосредственно в строку. Обычно это считается наиболее читаемым способом форматирования строк в C#.

Console.ReadLine

Метод Console.ReadLine используется для чтения строки текста из консольного ввода. Он возвращает введенные данные в виде строки.

Синтаксис

string input = Console.ReadLine();

Пример

using System;

public class Example
{
    public static void Main(string[] args)
    {
        Console.WriteLine("Пожалуйста, введите ваше имя:");
        string name = Console.ReadLine(); // Читает имя, введенное пользователем
        Console.WriteLine("Привет, " + name + "!");
    }
}

Объяснение

  • Код сначала предлагает пользователю ввести свое имя с помощью Console.WriteLine.
  • Затем Console.ReadLine ожидает, пока пользователь что-нибудь напечатает и нажмет Enter. Введенный текст затем сохраняется в переменной name в виде строки.
  • Наконец, код приветствует пользователя по имени с помощью Console.WriteLine.

Комбинирование WriteLine и ReadLine

Вы можете комбинировать WriteLine и ReadLine для создания интерактивных консольных приложений.

Пример

using System;

public class Example
{
    public static void Main(string[] args)
    {
        Console.WriteLine("Введите первое число:");
        string num1Str = Console.ReadLine();
        int num1 = int.Parse(num1Str);

        Console.WriteLine("Введите второе число:");
        string num2Str = Console.ReadLine();
        int num2 = int.Parse(num2Str);

        int sum = num1 + num2;

        Console.WriteLine("Сумма равна: " + sum);
    }
}

Объяснение

  • Код предлагает пользователю ввести два числа.
  • Он использует Console.ReadLine для чтения чисел в виде строк.
  • Поскольку ReadLine возвращает строку, нам нужно преобразовать строки в целые числа с помощью int.Parse(), прежде чем мы сможем выполнить сложение. Обратите внимание: важно обрабатывать возможные исключения, которые могут быть вызваны, если пользователь вводит нечисловые данные. См. следующий раздел.
  • Наконец, код вычисляет сумму и отображает ее в консоли.

Обработка ошибок с ReadLine

Крайне важно включать обработку ошибок при использовании Console.ReadLine и таких методов, как int.Parse(), поскольку пользователи могут вводить неожиданные данные. Использование int.TryParse может предотвратить сбой вашей программы.

Пример

using System;

public class Example
{
    public static void Main(string[] args)
    {
        Console.WriteLine("Введите первое число:");
        string num1Str = Console.ReadLine();
        int num1;

        if (!int.TryParse(num1Str, out num1))
        {
            Console.WriteLine("Неверный ввод для первого числа. Пожалуйста, введите целое число.");
            return; // Выход из программы, если ввод недействителен
        }

        Console.WriteLine("Введите второе число:");
        string num2Str = Console.ReadLine();
        int num2;

        if (!int.TryParse(num2Str, out num2))
        {
            Console.WriteLine("Неверный ввод для второго числа. Пожалуйста, введите целое число.");
            return; // Выход из программы, если ввод недействителен
        }

        int sum = num1 + num2;

        Console.WriteLine("Сумма равна: " + sum);
    }
}

Объяснение

  • int.TryParse() пытается преобразовать входную строку в целое число.
  • Если преобразование выполнено успешно, он возвращает true, и преобразованное целое число сохраняется в переменных num1 и num2.
  • Если преобразование не удается (например, пользователь вводит текст вместо числа), он возвращает false и отображается сообщение об ошибке. Затем программа завершает работу, чтобы предотвратить дальнейшие ошибки.
