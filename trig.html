Тригонометрия

Здесь вы найдете все необходимые конспекты по тригонометрии.

Разделы:

1) Основные понятия тригонометрии

Тригонометрия изучает соотношения между сторонами и углами треугольников, а также тригонометрические функции (синус, косинус, тангенс, котангенс) и их свойства.

  • Угол: Величина поворота. Измеряется в градусах или радианах.
  • Радиан: Угол, при котором длина дуги окружности равна радиусу окружности. 1 радиан ≈ 57.3 градуса.
  • Треугольник: Геометрическая фигура с тремя сторонами и тремя углами.
  • Прямоугольный треугольник: Треугольник, один из углов которого равен 90 градусам.
  • Гипотенуза: Сторона прямоугольного треугольника, лежащая напротив прямого угла.
  • Катеты: Две стороны прямоугольного треугольника, прилежащие к прямому углу.
2) Тригонометрические функции острого угла

Для острого угла α в прямоугольном треугольнике:

  • Синус (sin α): Отношение противолежащего катета к гипотенузе. sin α = (противолежащий катет) / (гипотенуза)
  • Косинус (cos α): Отношение прилежащего катета к гипотенузе. cos α = (прилежащий катет) / (гипотенуза)
  • Тангенс (tan α): Отношение противолежащего катета к прилежащему. tan α = (противолежащий катет) / (прилежащий катет) = sin α / cos α
  • Котангенс (cot α): Отношение прилежащего катета к противолежащему. cot α = (прилежащий катет) / (противолежащий катет) = cos α / sin α = 1 / tan α

Важные значения: sin 30° = 1/2, cos 60° = 1/2, tan 45° = 1, и т.д.

3) Тригонометрические функции произвольного угла

Тригонометрические функции определяются для углов от 0° до 360° (или от 0 до 2π радиан) и далее, с использованием единичной окружности:

  • Единичная окружность: Окружность с радиусом 1, центр которой находится в начале координат.
  • sin α: Ордината точки на единичной окружности, соответствующей углу α.
  • cos α: Абсцисса точки на единичной окружности, соответствующей углу α.
  • tan α: sin α / cos α (определена для всех α, кроме cos α = 0, т.е. для углов, кратных 90°).
  • cot α: cos α / sin α (определена для всех α, кроме sin α = 0, т.е. для углов, кратных 180°).

Четверти: Определяют знаки тригонометрических функций в каждой четверти единичной окружности (I, II, III, IV).

4) Основные тригонометрические тождества
  • Основное тригонометрическое тождество: sin² α + cos² α = 1
  • 1 + tan² α = 1 / cos² α
  • 1 + cot² α = 1 / sin² α
  • Формулы приведения: Определяют, как выразить тригонометрические функции углов, отличающихся от α на кратные 90° (π/2). Например:
    • sin(90° - α) = cos α
    • cos(90° - α) = sin α
5) Тригонометрические функции суммы и разности углов
  • sin(α + β) = sin α cos β + cos α sin β
  • sin(α - β) = sin α cos β - cos α sin β
  • cos(α + β) = cos α cos β - sin α sin β
  • cos(α - β) = cos α cos β + sin α sin β
  • tan(α + β) = (tan α + tan β) / (1 - tan α tan β)
  • tan(α - β) = (tan α - tan β) / (1 + tan α tan β)
6) Тригонометрические функции двойного и тройного угла
  • Двойной угол:
    • sin 2α = 2 sin α cos α
    • cos 2α = cos² α - sin² α = 2cos² α - 1 = 1 - 2sin² α
    • tan 2α = (2 tan α) / (1 - tan² α)
  • Тройной угол:
    • sin 3α = 3sin α - 4sin³ α
    • cos 3α = 4cos³ α - 3cos α
    • tan 3α = (3tan α - tan³ α) / (1 - 3tan² α)
7) Преобразование суммы тригонометрических функций в произведение
  • sin α + sin β = 2 sin((α + β) / 2) cos((α - β) / 2)
  • sin α - sin β = 2 cos((α + β) / 2) sin((α - β) / 2)
  • cos α + cos β = 2 cos((α + β) / 2) cos((α - β) / 2)
  • cos α - cos β = -2 sin((α + β) / 2) sin((α - β) / 2)
  • tan α + tan β = sin(α + β) / (cos α cos β)
8) Решение треугольников
  • Теорема синусов: a / sin A = b / sin B = c / sin C (где a, b, c - стороны треугольника, A, B, C - противолежащие им углы).
  • Теорема косинусов: c² = a² + b² - 2ab cos C
  • Решение треугольника: Нахождение всех сторон и углов треугольника, если известны некоторые из них (например, две стороны и угол между ними, или три стороны).
9) Обратные тригонометрические функции
  • Арксинус (arcsin x): Функция, обратная к синусу. arcsin x = y, если sin y = x. Область значений: [-π/2, π/2].
  • Арккосинус (arccos x): Функция, обратная к косинусу. arccos x = y, если cos y = x. Область значений: [0, π].
  • Артангенс (arctan x): Функция, обратная к тангенсу. arctan x = y, если tan y = x. Область значений: (-π/2, π/2).
  • Арккотангенс (arccot x): Функция, обратная к котангенсу. arccot x = y, если cot y = x. Область значений: (0, π).
10) Решение тригонометрических уравнений

Тригонометрические уравнения содержат тригонометрические функции. Решение таких уравнений часто включает в себя использование тригонометрических тождеств, формул приведения и знания областей значений и периодов тригонометрических функций.

Примеры простейших уравнений:

  • sin x = a Решения: x = (-1)^k arcsin a + πk, где k - целое число.
  • cos x = a Решения: x = ± arccos a + 2πk, где k - целое число.
  • tan x = a Решения: x = arctan a + πk, где k - целое число.
  • cot x = a Решения: x = arccot a + πk, где k - целое число.
